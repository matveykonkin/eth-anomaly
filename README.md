**Обнаружение аномалий в транзакциях Ethereum кошельков Binance**


## Описание

Проект для анализа и обнаружения аномальных транзакций в данных кошельков Binance с использованием методов машинного обучения.

## Структура проекта

eth-anomaly/
├── data/
│ ├── raw/ # Исходные данные
│ └── processed/ # Обработанные данные
├── notebooks/ # Jupyter ноутбуки для исследования
├── src/
│ ├── data/ # Скрипты загрузки и обработки данных
│ ├── visual/ # Визуализация данных
│ └── models/ # Реализация моделей для сравнения
└── README.md

## Промежуточный результат

# Файл src/data/data_loader.py загружает данные о транзакциях с указанного кошелька {address} через Ethercsan, в файл {data/raw/etherscan_data.csv}
# Файл src/data/data_processing.py обрабатывает данные из файла {data/raw/etherscan_data.csv}, переводит значения в формат необходимый для анализа и загружает их в файл {data/processed/features.csv}
# В директории src/visual находится скрипт для визуализации обработанных данных и сами графики
# Было загруженно первые 3000 операций с горячего кошелька Binance, обработаны и создано два простых графика для первичного анализа
