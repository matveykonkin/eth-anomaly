**Обнаружение аномалий в транзакциях Ethereum кошельков Binance**


## Описание

Проект для анализа и обнаружения аномальных транзакций в данных кошельков Binance с использованием методов машинного обучения.

## Структура проекта

eth-anomaly/
├── data/
│ ├── raw/ # Исходные данные
│ └── processed/ # Обработанные данные
├── notebooks/ # Jupyter ноутбуки для исследования
├── src/
│ ├── data/ # Скрипты загрузки и обработки данных
│ ├── visual/ # Визуализация данных
│ └── models/ # Реализация моделей для сравнения
└── README.md
